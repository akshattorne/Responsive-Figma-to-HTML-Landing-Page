# Responsive Figma to HTML Landing Page

A pixel-perfect responsive landing page built from a Figma design using **HTML5, CSS3, and JavaScript**. This project demonstrates responsive web design, semantic HTML, clean CSS architecture, and an interactive mobile navigation drawer while maintaining close visual fidelity to the original Figma design.
